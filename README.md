Airline Performance & Delay Analysis

Description: Engineered a high-performance BI solution analyzing over 2 million flight records to pinpoint delay root causes. Leveraging a robust Star Schema data model and Power Query for complex ETL, this dashboard provides granular insights into carrier reliability, weather impacts, and airport bottlenecks to optimize operational decisions.
